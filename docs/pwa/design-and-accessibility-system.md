# Design and accessibility system

The visual language uses calm dark neutrals, warm paper surfaces, and restrained mint and amber accents. Evidence dimensions always include words and semantic markup; color and icons are supplemental. The UI is mobile-first: the 320px single-column, touch-operated experience is the base layout and wider `min-width` queries add space and columns without changing workflow meaning. It supports zoom, reduced motion, forced colors, keyboard navigation, visible focus, skip links, and stable heading order.

External signed text is inert, bidirectionally isolated plain text. Dialogs trap focus, restore it on close, and name destructive scope. Progressive evidence uses restrained live announcements. Long authorities and hashes are expandable and copyable without dominating the primary flow.
